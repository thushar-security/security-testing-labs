# Security Testing & VAPT Labs

This repository contains hands-on security testing labs performed for learning and practice.

## Skills Demonstrated
- Web Application Security Testing
- OWASP Top 10
- Vulnerability Assessment & Penetration Testing (VAPT)
- Network Scanning & Enumeration
- Secure Data Sanitization

## Tools Used
- Burp Suite
- Nmap
- DVWA
- OWASP Juice Shop
- BLANCCO
- KillDisk
